<p align="center" style='font-size: 25px; text-decoration: underline; font-weight: bold'>Descrição:</p>

Projeto criado para mostrar a previsão do tempo, utilizando uma API chamada [weatherAPI](https://www.weatherapi.com/). Mostra informações como: Temperatura, umidade, velocidade do tempo e Status.

Foi criado utilizando: React, TypeScript, Tailwind, Axios e Chakra UI.

Teste o projeto [aqui](https://joao-vtr-oliveira.github.io/weather-API/).
